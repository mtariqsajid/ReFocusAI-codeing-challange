## Getting Started

### Prerequisites

- Node.js v20.18.1
- npm, yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/
cd frontend
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features in Detail
